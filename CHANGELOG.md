# 更新记录

- 建立增长总览、灵感收件箱、内容工作台、复盘与对标四个页面。
- 支持内容看板和表格两种视图。
- 发布登记要求真实平台、时间、结果和至少一种发布凭证。

- 增加本地状态 API、文件变化通知和网页自动回读。
- 增加跨平台环境检查、首次初始化和受控 Agent CLI。
- 增加 Codex、Claude Code 可直接执行的 AI 安装协议。
- 增加 Windows、macOS、Linux 的 GitHub Actions 验证矩阵。
